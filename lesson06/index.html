Селектори

A - як селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.

#id - ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.

.class - застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково або повністю схожі елементи оформлення. У коді HTML ім'я класу визначається через атрибут class.

* - іноді потрібно встановити одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт або накреслення тексту. У цьому випадку допоможе універсальний селектор, який відповідає будь-якому елементу веб-сторінки.

active - вибірка активного посилання.

:link - псевдоклас для стилізації посилань, на які користувач ще не натискав.

:visited - псевдоклас для стилізації тільки тих посилань, на які користувач натиснув або відвідав

:focus - вибір елемента введення, що має фокус

:link - вибір усіх відвіданих посилань

element - вибір усіх елементів
